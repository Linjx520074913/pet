# Pet Connect - 功能使用指南

## 🎯 应用概述

Pet Connect 是一个功能完整的宠物社交与管理应用，包含宠物配对、聊天、以及完整的宠物管理系统。

## 🚀 核心功能

### 1. 用户认证流程
- **启动画面** → 展示品牌Logo，2秒后自动跳转
- **引导页** → 3页滑动介绍，可跳过
- **登录页** → 邮箱密码登录 + Google/Apple社交登录
- **验证页** → 4位数字验证码输入

### 2. 主界面导航
底部导航栏包含4个标签：
- **Home（首页）** - 宠物发现与配对
- **Messages（消息）** - 聊天列表
- **Favorites（收藏）** - 待实现
- **Profile（个人）** - 用户资料与设置

### 3. 宠物发现（Home）
#### 特色功能
- 精美的宠物卡片展示
  - 网络图片加载（来自Unsplash）
  - 渐变遮罩效果
  - 年龄徽章
  - 特性标签（已接种、友好等）
- 滑动交互
  - ❌ 不喜欢（红色按钮）
  - ⭐ 超级喜欢（蓝色按钮）
  - ❤️ 喜欢（黄色按钮）
- 卡片切换动画
  - 淡出效果
  - 缩放动画
- 7只示例宠物数据

### 4. 消息系统
#### 消息列表
- 在线状态指示器（绿点）
- 未读消息计数徽章
- 最后消息预览
- 时间戳显示

#### 聊天界面
- 实时聊天UI
- 消息气泡（发送/接收）
- 文件附件按钮
- 发送按钮
- 在线状态显示

### 5. 个人资料
#### 展示内容
- 用户头像与信息
- 统计数据
  - 配对数量
  - 约会次数
  - 评分
- 我的宠物卡片
- 菜单项
  - 编辑资料
  - **管理宠物** ⭐
  - 位置管理
  - 通知设置
  - 隐私设置
  - 帮助支持
  - 退出登录

## 🐾 宠物管理系统（核心功能）

### 1. 我的宠物列表
**路径**: Profile → Manage Pets

#### 功能
- **标签筛选**
  - 全部（All）
  - 狗（Dogs）
  - 猫（Cats）
- **宠物卡片**
  - 80x80 圆角图片
  - 姓名与品种
  - 性别标识（蓝色/粉色）
  - 年龄与体重信息芯片
- **操作**
  - 点击查看详情
  - FAB按钮添加新宠物

### 2. 宠物详情页
**路径**: My Pets → 点击宠物卡片

#### 展示内容
- **顶部大图**
  - Hero动画过渡
  - 渐变遮罩
- **基本信息**
  - 姓名（大号粗体）
  - 品种
  - 编辑按钮
- **统计网格**
  - 年龄（图标+文字）
  - 体重
  - 性别
  - 疫苗状态
- **特性标签**
  - 黄色圆角徽章
  - 如：Friendly, Energetic, Playful
- **关于描述**
  - 多行文本
  - 详细介绍
- **宠物护理快捷入口**
  - 🍽️ Feeding（喂养）
  - 📖 Diary（日记）
  - ❤️ Health（健康）
- **最近活动**
  - 时间轴展示
  - 图标+标题+时间

### 3. 添加宠物
**路径**: My Pets → 点击 + 按钮

#### 表单内容
1. **照片上传**
   - 120x120 圆形头像框
   - 相机图标按钮

2. **基本信息**
   - 宠物名称 *必填
   - 宠物类型（Dog/Cat）*选择
   - 品种
   - 性别（Male/Female）*选择
   - 出生日期 *日期选择器
   - 体重（kg）

3. **详细描述**
   - 关于（多行输入）

4. **健康信息**
   - 已接种疫苗（开关）
   - 已绝育（开关）

5. **性格特征**
   - 10个可选标签
   - 多选模式
   - 选中变黄色背景

#### 验证
- 名称必填验证
- 提交后显示成功提示

### 4. 喂养日志

- FAB添加按钮

### 5. 宠物日记

#### 功能

## 📱 使用流程示例

### 场景1: 添加新宠物
1. 打开应用 → 登录
2. 底部导航 → Profile
3. 点击 "Manage Pets"
4. 点击右下角 FAB "Add Pet"
5. 填写表单：
   - 上传照片
   - 输入名称 "Max"
   - 选择类型 "Dog"
   - 输入品种 "Golden Retriever"
   - 选择性别 "Male"
   - 选择生日
   - 输入体重 30.5
   - 打开"已接种疫苗"
   - 选择特征：Friendly, Playful, Energetic
6. 点击 "Add Pet"
7. 看到成功提示，返回列表

### 场景2: 记录喂养
1. My Pets → 点击 Max
2. Pet Care → 点击 "Feeding"
3. 点击右下角 + 按钮
4. 底部弹窗填写：
   - 食物类型: "Dog Food - Premium"
   - 数量: 200
   - 单位: g
   - 备注: "Breakfast"
5. 点击 "Add Record"
6. 看到新记录添加到列表顶部

### 场景3: 写日记
1. Pet Detail → 点击 "Diary"
2. 点击 "Write" 按钮
3. 填写：
   - 标题: "First Day at Park"
   - 内容: "Max had so much fun today..."
   - 心情: 😊 Happy
4. 点击 "Save Entry"
5. 日记添加到时间轴
