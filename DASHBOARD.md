# 🏠 Dashboard - 宠物管理主屏幕

## 概述

全新设计的主屏幕（Dashboard）将**宠物管理**作为核心功能，提供一站式宠物护理体验。

## 🎯 主要功能

### 1. 我的宠物区域
**位置**: 顶部区域

#### 功能
- **水平滚动卡片**
  - 显示所有宠物的快捷卡片
  - 160x200 精美卡片设计
  - Hero 动画跳转到详情页
  - 点击即可查看宠物详情

- **添加宠物卡片**
  - 虚线边框设计
  - 显眼的 + 图标
  - 一键跳转到添加页面

- **查看全部按钮**
  - 右上角"View All"链接
  - 跳转到完整宠物列表

#### 卡片内容
- 宠物照片（120px高度）
- 宠物名称（粗体）
- 年龄 + 品种信息
- 圆角阴影设计

### 2. 快捷操作区域
**位置**: 中部，2x2网格布局

#### 4个快捷按钮
1. **🍽️ Feeding（喂养）**
   - 黄色主题
   - 直达喂养记录页
   - 默认选择第一只宠物

2. **📖 Diary（日记）**
   - 青色主题
   - 直达宠物日记
   - 记录美好时刻

3. **❤️ Health（健康）**
   - 红色主题
   - 健康档案管理
   - 疫苗&检查记录

4. **🐾 All Pets（所有宠物）**
   - 绿色主题
   - 查看完整宠物列表
   - 管理所有宠物

#### 设计特点
- 圆形图标背景
- 半透明颜色主题
- 卡片式布局
- 带边框高亮

### 3. 今日任务
**位置**: 中下部

#### 功能
- **任务列表展示**
  - 喂养任务
  - 遛狗任务
  - 医疗预约
  - 其他待办事项

- **任务状态**
  - ⭕ 未完成（空心圆）
  - ✅ 已完成（绿色对勾）
  - 完成任务带删除线

#### 任务卡片包含
- 彩色图标
- 任务标题
- 副标题说明
- 时间显示
- 完成状态

### 4. 发现新朋友
**位置**: 底部

#### 功能
- **渐变卡片设计**
  - 黄色→青色渐变
  - 大号宠物图标
  - 引导文案

- **一键跳转**
  - 点击"Explore"按钮
  - 跳转到宠物发现页面
  - 原有的配对功能

## 📱 使用流程

### 场景1: 快速喂养记录
```
1. 打开应用（自动进入Dashboard）
2. 点击"Feeding"快捷按钮
3. 点击 + 添加记录
4. 填写食物和数量
5. 保存完成
```

### 场景2: 查看宠物详情
```
1. Dashboard首页
2. 滚动"My Pets"区域
3. 点击想查看的宠物卡片
4. Hero动画跳转到详情页
5. 查看完整信息
```

### 场景3: 添加新宠物
```
1. Dashboard首页
2. 点击"Add Pet"虚线卡片
   或
   点击快捷操作中的"All Pets"
3. 进入添加页面
4. 填写表单
5. 保存完成
```

### 场景4: 探索新朋友
```
1. Dashboard首页
2. 滚动到底部"Discover New Friends"
3. 点击"Explore"按钮
4. 进入宠物发现页面
5. 开始滑动配对
```

## 🚀 技术实现

### 组件结构
```
DashboardScreen
├── SliverAppBar (可折叠顶栏)
├── My Pets Section (横向滚动)
├── Quick Actions (2x2网格)
├── Today's Tasks (列表)
└── Discover Section (渐变卡片)
```

### 关键代码
- `CustomScrollView` - 滚动容器
- `SliverAppBar` - 折叠顶栏
- `Hero` - 页面跳转动画
- `Navigator.push` - 路由导航
